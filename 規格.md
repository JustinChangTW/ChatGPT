# 規格.md（給 Codex 的實作規格）

## A. 角色與權限
- admin：可匯入題庫、改系統參數、看全體進度。
- member：可作答、用筆記、看錯題。
- guest：公共內容只讀（依 rules）。

## B. 筆記規格
- 每題有兩種筆記：
  1) 共編筆記（shared）
  2) 私人筆記（private）
- 錯題本展開題目時，必須顯示 shared + private。

## C. 模擬考規格
- 產生試券即建立紀錄。
- 作答中持續更新該試券紀錄。
- 交卷後更新為最終成績。

## D. Codex 驗收命令
- `npm run -s lint`
- `npm run -s build`（頁面行為有變動時）
- `npm run -s typecheck`（若失敗要在回覆註明）
