# 規格.md（現行功能規格）

## 1. 角色與權限
- `admin`
  - 可匯入題庫
  - 可調整 API/AI 系統參數
  - 可查看全體使用者作答概況（Admin 後台）
- `member`
  - 可作答、查看錯題、使用 AI 助教
  - 可使用共編筆記與私人筆記
- `guest`
  - 可瀏覽公共內容（依 Firestore rules）

## 2. 筆記系統
### 2.1 共編筆記
- 以題目為單位共享。
- 任一使用者可查看（規則依 Firestore）。

### 2.2 私人筆記
- 以「使用者 + 題目」為單位存放。
- 僅本人可見（目前以本機儲存層保證）。

### 2.3 錯題本展示
- 錯題展開後需顯示：
  - 共編筆記
  - 私人筆記（當前使用者）

## 3. 練習/模擬考
### 3.1 章節練習
- 支援逐題即時覆盤與清單模式。
- 筆記可在章節練習編輯並分別儲存（共編/私人）。

### 3.2 模擬考
- 產生試券即建立紀錄。
- 作答中持續更新同筆試券紀錄。
- 交卷後轉為最終成績紀錄。

## 4. Admin
- 顯示目前 UID。
- 提供複製 UID 與複製 `users/{uid}` 路徑。
- 提供角色設定教學。

## 5. 驗證基準
- `npm run -s lint` 必須通過。
- `npm run -s build` 需可完成。
- `npm run -s typecheck` 若被既有測試 fixture 阻塞，需在 PR 註明。
